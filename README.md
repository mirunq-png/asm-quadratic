## Prerequisites
To run this project, you will need:

* Visual Studio Code
* The MASM/TASM extension for VS Code

## How to Run
1) Clone this repository to your local machine:
   
```git clone https://github.com/mirunq-png/asm-quadratic```

2) Open the project folder in Visual Studio Code.
4) Open the ```quadratic.asm``` file in the editor.
5) Right-click anywhere inside the code editor and select ```Run ASM code``` from the context menu.

## Features
* Solves equations of the form $ax^2+bx+c=0$ using custom 16-bit integer division and arithmetic.
* Custom implementation of the Babylonian method to calculate square roots.
